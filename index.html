How authentication works in this app:
Login is handled by the app's own User Management system — Firebase Authentication is not used and not needed. Users and passwords are managed in Admin Panel → User Management. Firebase here is used only for syncing data (Realtime Database) across devices.

Only the Realtime Database URL is required. You do not need to enable Firebase Authentication.

Realtime Database Rules

Paste these in Firebase Console → Realtime Database → Rules.

{ "rules": { ".read": true, ".write": true } }
